jQuery to read and display xml data

Fetching data stored in XML file manually.

This experiment demonstrates how data can be fetched from xml file that is stored by the user in the server. Here, the personal details are launched from student_catalog.xml that I've stored in the server.


Code Snippet
function displaySTUDENT()
{
name=(x[i].getElementsByTagName("NAME")[0].childNodes[0].nodeValue);
city=(x[i].getElementsByTagName("CITY")[0].childNodes[0].nodeValue);
age=(x[i].getElementsByTagName("AGE")[0].childNodes[0].nodeValue);
txt="Name: " + name + "
City: " + city + "
Age: "+ age;
document.getElementById("showSTUDENT").innerHTML=txt;
}
