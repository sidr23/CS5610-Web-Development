Retrieving live cricketing news using RSS feed data

This experiment demonstrates how data can be fetched from xml file using RSS. Here, the news is launched from a cricketing website and to see further details one can click on the detailed report link to take them to the parent site i.e. Cricinfo.


Code Snippet
