Dropdown Using JQuery and CSS
In this experiment, I have implemented the dropdown menu. 
We need to hover over the menu and its submenus drop down
Javascript code:
$(function () 
{
$('nav li ul').hide().removeClass('sublevel');
$('nav li').hover(function () {
$('ul', this).stop().slideToggle(200);
});
});
