Magnify image using CSS,JQuery


Zooming an image by CSS,Jquery.

This effect is usually seen in Apple's site and can be used in web pages to magnify details.
The factor of magnification in this image is very low because of the size and relative coordinates formula used.

Code Snippet:
var rx = Math.round(mx/$(".small").width()*native_width - $(".large").width()/2)*-1;
var ry = Math.round(my/$(".small").height()*native_height - $(".large").height()/2)*-1;
var bgp = rx + "px " + ry + "px";
var px = mx - $(".large").width()/2;
var py = my - $(".large").height()/2;

Hover mouse to magnify the image.
