Auto Complete feature using JQuery

This experiment explores the autocomplete feature of jQuery. I have made a list of sports.
As the user starts typing the name in the input field, a list of related names appears and the user can select from that list.
At the bottom of the page a label indicates what was searched and its results





The following sports were used:
Cricket
Hockey
Football
Soccer
Tennis
Golf
Basketball
