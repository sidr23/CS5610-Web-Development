Loading content from a file using Ajax

The textfiles 1.txt has text in it.
Instead of static content, AJAX allows fetching of data from files into the webpage which makes it easier to users who don't know to code in HTML.
This allows the developer to provide just the outline like this HTML file and the text can be changed by the user as per his specifications.
If the file is deleted or not there, it automatically prints not found.

Displaying content off a text file

Code Snippet:


$(document).ready(function(){
$("button").click(function(){
$("#div1").load("1.txt", function(responseTxt, statusTxt, xhr){
if(statusTxt == "success")
alert("Content loaded successfully from file!");
if(statusTxt == "error")
alert("Error: " + xhr.status + ": " + xhr.statusText);
});
});
});
