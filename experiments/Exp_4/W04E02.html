Ajax to fetch scores

Fetching live score from an API that has live cricket scores. This is done entirely using Javascript,Ajax and Jquery.
Click on the Live Score Button to retrieve scores. json-cricket appspot stores cricket scores in json

Code Snippet:

function test() {
$.ajax({
url: "http://json-cricket.appspot.com/score.json",
dataType: "jsonp",
success: function (data) {
$("#scores").empty();
$.each(data, function (i, item) {
$("#scores").append(item);
});
}});
}
