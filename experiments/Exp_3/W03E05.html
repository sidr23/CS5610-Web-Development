Accordian sortable is a common feature in JQuery and has been provided by JQuery.com as an open source tool. It can be freely downloaded and applied to produce such effects! Followed the video and downloaded the JS file from JQuery.com. Easily applied the styles that were previously defined in it. Mouse Click over the list items to see the hidden submenus!

JQuery Code Snippet:

$(function ()
{
$("#accordion")
.accordion({
header: "> div > h3"
})
.sortable({
axis: "y",
handle: "h3",
stop: function (event, ui)
{
$(this).accordion("refresh");
}
});
});
